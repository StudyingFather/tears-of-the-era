# 时代的眼泪

「[时代的眼泪](./docs/noi2020-tears.html)」是 NOI2020 D1T3 的题目名。

据出题人说，因为 OI 中较少涉及分块科技，如同时代的眼泪一般，故以「时代的眼泪」作为题目的名字。

> A：2020 年怎么会考根号数据结构？
>
> B：是啊，所以这玩意已经成为了**时代的眼泪**。

本站计划成为一个 OI 历史存档站点，通过对历史资料的整理，向更多人展现 OI 的发展历程。
